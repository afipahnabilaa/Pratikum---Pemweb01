Fulan Tertangkap Basah Ngoding Jam 3 Pagi Demi Push ke GitHub

Jakarta, Portal Fulan- Warga sekitar dikejutkan dengan suara ketikan keyboard yang tak kunjung berhenti. Setelah diselidiki, ternyata sumber suara berasal dari kamar Fulan yang sedang men-deploy project-nya ke Vercel

"Aku cuma mau memastikan web-nya responsif sebelum tidur", ujar Fulan sambil menyeruput kopi dingin yang udah cair dar jam 10 malam.

Menurut pengakuan saksi, Fulan sudah tiga malam tidak tidur gara-gara memperbaiki error vite-pligin-pwa missing. Namun akhirnya project berhasil jalan dan langsung mendapat 5 ‚≠ê di GitHub dari teman satu kos-nya.

Fakta Coding Fulan

  • Suka debugging sambil dengerin musik anime
  • Pernah typo console.log jadi consola.lag
  • Setiap error dianggap "ujian cinta terhadap sintaks"

Top 3 Error Paling Sering Ditemui

  1. Missing semicolon (;) setelah jam 2 pagi
  2. Path image tidak terbaca di deploy
